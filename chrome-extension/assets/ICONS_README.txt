ICON PLACEHOLDERS NEEDED

The extension requires the following icon files:
- icon16.png (16x16 pixels)
- icon48.png (48x48 pixels)
- icon128.png (128x128 pixels)

For now, you can create simple placeholder icons using any image editor,
or use the Vanish logo from /images/ directory if available.

Quick way to create placeholders with ImageMagick (if installed):
  convert -size 16x16 xc:#EF4444 icon16.png
  convert -size 48x48 xc:#EF4444 icon48.png
  convert -size 128x128 xc:#EF4444 icon128.png

Or download icons from: https://www.iconfinder.com/ or https://www.flaticon.com/
